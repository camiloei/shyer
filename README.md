# vuni
micro 2d game library

Current lib size: 156 LoC - 5kb (unminified)

Current Features:
  - Sprites
  - Camera
  - Scenes
  - Events
  - Input
  - Asset Loading

Future Features:
  - Animated Sprites
  - Tilemaps
  - Audio Playback / Events
  - Npm module setup 
  - WebGL Rendering (?)
  - Hardcore Perf Optimizations (?)
  
(?): not in the short term

Requirements:

1. yarn   
 ```bash

    curl -o- -L https://yarnpkg.com/install.sh | bash

  ```
Setup:

**build** 
```bash
'yarn build' .  
```
**run** 
```bash
'yarn start:dev'
```
